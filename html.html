Introduction to HTML

HTML (Hypertext Markup Language) is the backbone of web pages. It defines structure using tags like headings, paragraphs, links, and images.

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="url">Link</a>
  • Images: <img src="image.jpg" alt="desc">
